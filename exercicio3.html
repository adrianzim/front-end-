Lista de passos para criar uma página web simples

  1. Escolher um editor de HTML, como o Bloco de Notas, VS Code ou Sublime Text.
  2. Criar um novo arquivo com a extensão ".html".
  3. Escrever a estrutura básica do HTML, incluindo as tags <!DOCTYPE>, <html>, <head> e <body>.
  4. Adicionar conteúdo dentro do <body>, como títulos, parágrafos ou listas.
  5. Salvar o arquivo e abrir no navegador para visualizar a página.
